SIGCSE TS 2023 Nifty Assignment

Instructional Sequence to Target Common Misconceptions Held by Novice Programmers

Overview

An instructional sequence that consists of 16 isomorphic pairs of code tracing problems. The problems are designed to facilitate learning by teaching or inducing cognitive conflict.

Goals

To target common misconceptions held by novice programmers learning Java through formative feedback as a means to promote viable mental models.

Materials

A file formatted as Portable Document Format (.pdf), Microsoft Word Binary File Format (.doc), and Brightspace Package (.zip).

Metadata
Summary Each practice problem includes a short description, reference and url to the supporting misconceptions literature, question test, hint, four answer choices and relevant feedback, overall feedback in the form of on-screen text combined with either a memory diagram or expression tree.
Topics Variable and Data Types; Arithmetic Operations; String Operations; Control Flow Structures (Selection, Iteration); Data Structures (Arrays, Objects)
Audience Novice programmers in CS1 introductory courses using Java as a language.
Difficulty Intrinsic cognitive load is managed via short code snippets - no more than a few statements.
Strengths Problems are designed as isomorphic pairs to personalize instruction.
Weaknesses Problems should be used in combination with instructional content to support students in writing or revising code.
Dependencies Prior knowledge of the semantics of programming language features.
Variants Problems are aligned with common misconceptions from a repository curated by members of the SIGCSE community.
